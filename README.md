- Xiaomi R3G (Router 3G)
- Xiaomi R3P (Router 3 Pro)
- Xiaomi R3D (Router HD)
- Xiaomi R4  (Router 4)
- Xiaomi R2100 (Router AC2100)
- Xiaomi RM2100 (Redmi AC2100)
- Xiaomi RB01/RB03 (AX3200/AX6S)
- Xiaomi RB06 (Redmi AX6000)
- Xiaomi RD03/RD23 (AX3000T)
- ASUS RT-AX89X
- ASUS RT-AX52
- ASUS RT-AX57M
- ASUS RT-AX59U
- ASUS TUF-AX4200
- ASUS TUF-AX6000
- ASUS ZenWiFi-BT8
